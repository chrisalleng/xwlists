{% extends "layout.html" %} {% block content %} This website was built to help enter lists from live X-Wing tournaments. It was inspired by Major Juggler's tireless efforts to gather statistics around the various lists that we play. Please provide feedback to either sozin (the author) over on the Fantasy Flight Forums. If you're interested in contributing, the source code can be found over on github. The technical stack used to implement this site is:
  • Python for the server-side, using the excellent Flask framework
  • MySQL for the database, with SQL Alchemy for the object to relational mapping
  • Javascript for the client-side, using a variety of different Javascript libraries.
  • Pythonanywhere for hosting
  • The amazing Pycharm integrated development environment
